# AlGraphy Pro Hub 🚀

A premium link management platform (Linktree alternative) built with PHP (MVC), MySQL, and modern CSS.

## Features ✨
- **Dynamic Link Management**: Add, update, and reorder links with a drag-and-drop interface.
- **Social Media Integration**: Smart detection of social links (Instagram, X, TikTok, etc.) with automated icons.
- **Glassmorphism UI**: Beautiful, responsive profile pages with blurred backgrounds and neon accents.
- **Advanced Customization**: Live theme presets (Dark, Light, Ocean, Neon) and custom color pickers.
- **Analytics**: Track page views and link clicks (In-progress).
- **Responsive Dashboard**: Fully optimized for Mobile, Tablet, and Desktop.

## Tech Stack 🛠
- **Backend**: PHP 8.x (Custom MVC Architecture)
- **Database**: MySQL
- **Frontend**: Vanilla CSS, JavaScript, FontAwesome 6+
- **Security**: Password hashing, CSRF protection, and session management.

## Installation 💻
1. Clone the repository into your XAMPP `htdocs` folder.
2. Import `database.sql` into your MySQL server.
3. Configure your database connection in `app/Core/Database.php`.
4. Access via `http://localhost/algraphyHub`.
