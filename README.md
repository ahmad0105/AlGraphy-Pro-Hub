# 🚀 AlGraphy Pro Hub

**AlGraphy Pro Hub** is a premium, high-performance "Link-in-bio" platform designed for creators and businesses to centralize their digital presence in a sleek, fast, and highly secure landing page.

---

## ✨ Key Features

- **Intuitive Dashboard:** A modern administrative interface to manage profile links, avatars, and bios in real-time.
- **Dynamic Personalization:** Complete control over branding, including custom color palettes, modern typography, and diverse background types (Solid Colors, Images, or Video backgrounds).
- **Social Icon Management:** Integrated Drag & Drop system for reordering social media icons to prioritize your most important channels.
- **Advanced Analytics:** Track profile performance with real-time data on Total Views, Unique Visits, Link Clicks, and Click-Through Rate (CTR).
- **Instant QR Code:** Automatically generated QR codes for your profile, optimized for high-quality downloads and physical marketing materials.
- **Premium UX/UI:** Responsive design with Glassmorphism effects and micro-animations, optimized for mobile-first interactions.

---

## 🛠 Technology Stack

### **Frontend:**
- **HTML5 & CSS3:** Custom Design System built with Vanilla CSS, leveraging CSS Variables for themes and smooth transitions.
- **Vanilla JavaScript (ES6+):** Lightweight, dependency-free logic for modals, toasts, and live preview updates.
- **Font Awesome 6:** Comprehensive iconography for a professional look and feel.

### **Backend:**
- **PHP 8.2 (OOP Architecture):** Built on a robust Object-Oriented foundation with a clean MVC-inspired structure for maximum scalability.
- **PSR-4 Autoloading:** Professional file management and class loading.
- **MySQL/MariaDB:** Optimized relational database management for user profiles and analytics.

---

## 🛡 Security & Performance Optimization

Engineered with a "Security-First" mindset:
- **Hardened .htaccess:** Enhanced server-side security rules to block access to sensitive files (`.env`, `.git`, etc.) and prevent directory listing.
- **Content Security Policy (CSP):** Strict headers to mitigate XSS attacks and unauthorized script execution.
- **SVG Protection Layer:** Specific headers to prevent script execution within SVG assets, effectively neutralizing image-based XSS vulnerabilities.
- **Cross-Browser Compatibility:** Specialized URL generation logic to ensure seamless HTTPS/SSL performance across all modern browsers, including Safari.
- **Speed Optimization:** High-performance logo inlining (Base64) and intelligent resource preloading for near-instant page loads.

---

## 🚀 Installation & Deployment

1. **Environment Setup:** Upload files to a local server (XAMPP) or production hosting (e.g., InfinityFree).
2. **Database:** Import the provided `database.sql` file via phpMyAdmin.
3. **Configuration:** Update the `.env` file with your database credentials and `APP_URL`.
4. **Permissions:** Ensure the `logs/` directory is writable by the server.
